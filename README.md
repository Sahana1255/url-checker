# URL-CHECKER

A full-stack web application for comprehensive URL security analysis. Built with React (frontend) and Python Flask (backend).

## Project Structure

- `frontend/` — React + Vite client app, UI components, state management, and services
- `backend/` — Python Flask server, security analysis modules, database integration

## Features

- URL risk analysis
- Security headers check
- SSL certificate validation
- WHOIS domain analysis
- User authentication (JWT)
- Tiered subscriptions and feature access
- Report exporting
